HTML Tag

HTML (Hypertext Markup Language) uses heading tags to define the hierarchy and structure of headings on a webpage. There are six levels of heading tags in HTML: h1, h2, h3, h4, h5, and h6. In this response, I'll focus on describing the h1 to h3 tags.

Header

  1. h1: Header1 - Main Title of site or article. One per page
